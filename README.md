# Holzwerk – Hugo Site

Dieses Repository ist Teil eines **Studienprojekts** und dient zum Aufbau und Deployment der Website mit dem **Hugo Static Site Builder**.

## 📚 Projekt-Hintergrund

Im Rahmen des Studiums war die Aufgabe, **dieselbe Website viermal** mit unterschiedlichen Technologien und Ansätzen umzusetzen. Dieses Repo enthält die **Hugo-Variante** – eine der vier Implementierungen.

Ziel des Projekts war es, verschiedene Web-Technologien praktisch kennenzulernen, ihre Stärken und Schwächen zu vergleichen und ein Gefühl für unterschiedliche Entwicklungs-Workflows zu bekommen.

## 🛠️ Verwendete Technologie

- **[Hugo](https://gohugo.io/)** – ein schneller Static Site Generator, geschrieben in Go
- **GitHub Pages** – für das Hosting der Seite
- **GitHub Actions** – für den automatischen Build- und Deploy-Prozess
